Тестовое задание Текст в БД.
Создать веб-приложение для записи произвольных текстов в базу данных.

С каждым из текстов может быть записана (а может и не быть) некоторая дата.

Интерфейс приложения состоит из одного экрана, разделенного на 2 части.

В верхней части располагается поле для ввода произвольного текста, поле для ввода даты и кнопка "Записать".

В нижней части показывается до 3 последних записанных текстов. Если в базе данных записано более 3 текстов - показываются только последние 3.

Если пользователь не задает дату явно, но в тексте содержится указание на некоторый день, например:

8 марта
через неделю
вчера
в следующий вторник
и так далее (чем больше вариантов тем лучше)
То соответствующая дата должна сохраняться, как если бы пользователь ввел ее в поле даты явным образом.

Технологии, обязательные к использованию:

C#, ASP.NET MVC
HTML/CSS
MS SQL Server
Указанные технологии составляют минимальный набор, можно использовать и другие в дополнение к вышеназванным, на Ваше усмотрение.

Качество реализации должно быть максимально готово для использования в production, насколько Вы себе это представляете.
